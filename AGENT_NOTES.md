# Website Building - Agent Notes

**Project:** Website Building Workspace (Webinars, Carwashes, etc.)
**Status:** Active
**Version:** 1.0

## Overview

A directory acting as a workspace for various fast-paced web developments, including Austin Auto Spa (Carwash) and mastermind webinars. Often uses Vite/React/Tailwind stacks for rapid scaffolding.

## Quick Start

- Inspect subdirectories for project-specific stacks (e.g. `Carwash-Websites`, `brainchild-v2`).
- Check `design_system_output.md` for the current conversion-focused UI templates.
- Spline Skill available for 3D interactions.

## Architecture Overview

- Tends to use Node 20+, React, Tailwind, Vite.
- Spline 3D embeds for high-end interactives (React or Vanilla).
- Incorporates "Trust & Authority" design logic (Badges, dark mode + action red CTA #DC2626).

## Recent Learnings & Discoveries

- **Spline Skill Definition:** A well-documented skill folder (`🌍 Spline Skill`) added, which standardizes Spline.design integrations across Vanilla and React with fallback logic. Applicable to ANY frontend project needing 3D models.
